# 🥑 Avocado Market Analysis

##  Overview
This project analyzes avocado pricing and sales trends using Python.

## Objective
To understand how pricing, seasonality, and region impact demand.

## 🔧 Tools Used
- Python
- Pandas
- SQL

##  Key Insights
- Price negatively impacts demand
- Organic avocados are premium-priced
- Sales vary by region
- Seasonal trends affect pricing
